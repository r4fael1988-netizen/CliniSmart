💡 SE DER ERRO
═══════════════════════════════════════════════════════════════════════════════

O script vai dizer EXATAMENTE qual é o problema:

   "PostgreSQL: Não instalado localmente"
   → Use clinic_id=1 ou acesse via outra máquina

   "API retornou: 000"
   → API está offline, aguarde e tente novamente

   "N8N_API_KEY não foi definida"
   → Obtenha no n8n Settings → API Keys

   "Workflow não foi atualizado"
   → Faça manualmente (20 segundos no n8n)
